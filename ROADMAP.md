# Roadmap

SpecEval is in early alpha. Here's where we're headed.

## v0.2.0 — Stabilization

- [ ] Async model providers for parallel evaluation
- [ ] Streaming results to support long-running evals
- [ ] Custom metric authoring via Python plugins
- [ ] Better error messages in spec validation
- [ ] Pre-commit hook for spec linting

## v0.3.0 — Integrations

- [ ] Native Weights & Biases integration
- [ ] MLflow experiment tracking export
- [ ] HuggingFace dataset hub publishing
- [ ] GitHub Actions composite action for CI

## v0.4.0 — Advanced Comparison

- [ ] Multi-model comparison tables (N × N)
- [ ] Bootstrap confidence intervals on differences
- [ ] Bayesian model ranking (expected win probability)
- [ ] Equivalence testing (TOST)

## v1.0.0 — Stable Release

- [ ] Stable specification format (v1 schema)
- [ ] Comprehensive documentation site
- [ ] Performance benchmarks
- [ ] Migration guide from v0.x

## Ideas (not yet scheduled)

- Web dashboard for browsing results
- VS Code extension for spec files
- Self-hosted result server
- Automated regression detection

---

*This roadmap is a living document. Priorities shift based on feedback and contributions.*
