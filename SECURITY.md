# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in SpecEval, please report it privately by emailing **luvgoel555@gmail.com**.

Do **not** open a public issue for security vulnerabilities.

We aim to respond within 48 hours and will keep you informed of the resolution process.

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 0.1.x   | ✅        |

## Responsible Disclosure

We kindly ask that you give us a reasonable timeframe to address the issue before disclosing it publicly.
